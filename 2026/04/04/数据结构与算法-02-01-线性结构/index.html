线性表及其实现

线性表的定义

线性表：由同类型数据元素构成有序序列的线性结构。

  • 表中的元素个数称为线性表的长度
  • 线性表没有元素时候称为空表
  • 表起始位置称表头，表结束位置称表尾

线性表的抽象数据描述

用于帮助我们如何去设定这个线性表

  1. 类型名称：线性表(List)
  2. 数据对象集：线性表是n(>=0)个元素构成的有序序列(a1,a2,…………,an)
  3. 操作集：线性表L∈Last(某个类型)，整数i表示位置，元素X∈ElementType(某个类型)，线性表的基本操作如下：
    • List MakeEmpty() 初始化一个空线性表L
    • ElementType FindKth(int K,List L) 根据位序K，返回相应元素
    • int Find(ElementType X,List L) 在线性表L中，查找X的第一次出现位置
    • void Insert(ElementType X,int i,List L) 在位序i前插入一个元素x(也就是x成为位序i上的元素，原位序i及以后的均后移)
    • void Delete(int i,List L)删除指定位序i上的元素
    • int Length(List L)返回线性表L的长度n

顺序存储实现线性表

线性表顺序存储在物理意义上是连续的，是利用数组的连续存储空间顺序存放线性表的个元素

eg.

下标i 0 1 …… i-1 i …… n-1 …… MAXSIZE-1
Data a1 a2 …… ai ai+1 …… an …… -

其中的MAXSIZE是数组的最大值，而最后一个值就不一定是这个，所以我们是需要一个指针Last来指明最后一个元素在哪里的

示例:

typedef struct LNode* List;//创建一个新的数据类型叫List，之所以要创建，是为了简化书写，我们当然可以每次都写 Struct LNode *PtrL1;Struct LNode *PtrL2等等，但是每次都写，太麻烦了，不如创建一个新的数据类型，用List Ptrl1,List Ptrl2等等简化书写。
struct LNode{//定义一个结构体类型
    ElementType Data[MAXSIZE];
    int Last;
};
struct LNode L;/*声明一个LNode类型的变量L*/
List PtrL;

在这类，这段代码的思路是，

访问下标为i的元素：L.Data[i-1]或Ptrl->Data[i]
线性表的长度：L.Last+1或Ptrl->Last+1

接下来，我们讲阐述如何在顺序存储中实现操作集

初始化(建立空的顺序表)

typedef struct LNode* List;

struct LNode{
    ElementType Data[MAXSIZE];
    int Last;

List MakeEmpty(){
    List PtrL;
    Ptrl = (List)malloc(sizeof(struct LNode));
    Ptrl->Last = -1;//因为是空的，计算机从0开始计数，所以Last最开始应该为-1
    return Ptrl;
}

查找

int Find(ElementType X, List Ptrl){
    int i = 0;
    while(i <= Ptrl->Last && Ptrl->Data[i] != X){
        i++;
    }
    if (i > Ptrl->Last){
        return -1;//如果没找到就返回-1
    }
    else{
        return i;
    }
}

顺序存储的插入与删除

插入(在第i(1<=i<=n+1)个位置上插入一个值为X的新元素)
步骤
  • 移动
    • 先把每个元素都向后移动一位，直到第i个位置空出来(使用循环实现，要从最后一个开始挪哦)
  • 插入
例子
void insert(ElementType X, int i, List Ptrl)/*
注意，这里i表示的是位置，我们要说明插入位置不合法
i<1是指你不能插入第零个元素，你当然可以插入第一个元素，但是第零个是不可能的
i > Ptrl->Last+2是指你不可能插入到表的外面，Last是指下标，计算机从0开始计，所以Last+1是长度，那么Last+2就在表的外面，这是不可能的
*/
{
    int j;
    if(ptrl->Last == MAXSIZE-1){
        printf("表满");
        return;
    }
    if(i<1 || i > Ptrl->Last+2){
        printf("插入位置不合法");
        return;
    }
    for(j=Ptrl->Last;j>=i-1;j--){
        Ptrl->Data[j+1] = Ptrl->Data[j];
    }
    Ptrl->Data[i-1] = X;
    Ptrl->Last++;
    return;
}
}
删除(删除第i(1<=i<=n)个位置上的元素)
步骤
  • 移动，后面的元素依次前移一位
例子
void delete(int i, List Ptrl){
    int j;
    if(i<1 || i > Ptrl->Last+1){
        printf("删除位置不合法");
        return;
    }
    for(j=i-1;j<Ptrl->Last;j++){
        Ptrl->Data[j] = Ptrl->Data[j+1];
    }
    Ptrl->Last--;
    return;
}
